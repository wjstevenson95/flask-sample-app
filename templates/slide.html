jQuery - Slide

In this section, we will cover the jQuery slide methods, which allow the user to slide HTML elements up and down.

NOTE: This assumes that the user has a working knowledge of HTML, CSS, and JavaScript.

Using jQuery, there are three main methods that you will user to create a sliding effect on the HTML elements of the document.

  • slideDown()
  • slideUp()
  • slideToggle()

slideDown()

The slideDown() jQuery method is used to slide down an HTML element.

Under The Hood: Essentially, this method is animating the height attribute of HTML elements.

Syntax:

$(selector).slideDown(speed, callback)

  • The optional speed parameters specifies the duration of the effect.
  • The optional callback parameter is a function that will run as soon as the slide up effect as completed.
